# CloudCanvas Editor

CloudCanvas Editor is a browser-based image editing application built using HTML, CSS, and JavaScript.  
The project focuses on real-time image manipulation using the Canvas API and clean UI architecture.

## Features

- Real-time image filters (brightness, contrast, saturation)
- Preset-based adjustments
- Reset functionality
- Download edited image
- Responsive layout

## Tech Stack

- HTML5
- CSS3 (Custom Properties)
- JavaScript (Canvas API)

## Technical Focus

- Event-driven DOM manipulation
- State-based filter management
- Modular CSS structure
- Clean and scalable UI design
