# 🚀 CloudCanvas Editor

> A Cloud-Integrated Browser-Based Image Editing Application  
> Built with HTML, CSS, JavaScript & Firebase Authentication

---

## 🔥 Project Overview

CloudCanvas Editor is a responsive, browser-based image editing application that performs **real-time image manipulation using the HTML5 Canvas API**, now integrated with **Firebase Authentication** for secure user access.

This project reflects practical frontend engineering combined with cloud-backed authentication — demonstrating scalable architecture thinking and deployment readiness.

---

## ✨ Core Features

### 🎨 Image Processing
- Real-time Brightness, Contrast, Saturation controls
- Preset-based filter adjustments
- Instant preview using Canvas API
- Reset functionality
- Download edited image
- Fully responsive layout

### 🔐 Cloud Authentication
- Secure User Registration
- Login & Logout functionality
- Firebase Authentication integration
- Protected access workflow
